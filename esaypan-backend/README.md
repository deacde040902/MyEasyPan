# MyEasyPan
MyEasyPan 是我独立开发的仿百度网盘后端项目，基于 SpringBoot + Vue 前后端分离实现，主要用于练习 Java 全栈开发能力，同时实现了常见网盘的核心功能。

项目实现了文件分片上传、断点续传、秒传、在线预览、文件管理、回收站等完整逻辑，支持图片、音视频、PDF、Office 等多种格式预览。
用户模块包含注册、登录、QQ 登录、邮箱验证码等功能，同时支持带有效期和提取码的私密文件分享，他人可直接转存到自己网盘。
同时具有市面常见的通过alipay支付订单成为vip享有vip专属权限--更大的空间，更快的下载速率等。

整个项目从 0 到 1 完成需求分析、数据库设计、接口开发，熟练运用了 SpringBoot、MyBatis-Plus、MySQL、Redis 等技术栈。
适合作为个人学习、实战练习与求职展示的项目。
